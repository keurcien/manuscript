# manuscript

A couple of things need to be done beforehand. 

## Prerequisites

1. A LaTeX compiler
2. RStudio
3. python3

### Install bookdown

In a R session run

```
install.packages("bookdown", repos = "http://cran.rstudio.com")
```

### Install simuPOP

Use a terminal or command line and run

```
conda config --add channels conda-forge
conda install simupop
```
